# FitGPT Senior Symposium

QR code target URL:
https://fit-gpt-i3co.vercel.app/download

Android:
1. Scan the QR code.
2. Tap Download Android App.
3. Open the downloaded APK.
4. If Android asks, allow installs from this browser or file manager.
5. Install FitGPT.
6. Open the app and sign in or create an account.

iPhone:
1. Scan the QR code.
2. Tap Open Web Demo.
3. Use FitGPT in the browser.

APK file path:
app/build/outputs/apk/debug/app-debug.apk

Unsigned release APK path:
app/build/outputs/apk/release/app-release-unsigned.apk

Final APK download URL:
https://github.com/Muhammad7839/FitGPT/releases/download/symposium-demo-v1/FitGPT-Symposium-Demo.apk

Troubleshooting:
If installation does not work, use the web demo button. The web demo connects to the same live backend.
