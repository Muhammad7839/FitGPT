# Accessibility

## Overview
FitGPT provides accessibility options that allow users to customize the visual presentation of the app to better suit their needs. These settings are independent of the standard theme system and can be enabled or disabled at any time.

---

## Large Text Mode

### Description
Users can enable large text mode to scale up text across the app, improving readability for visually impaired users or those who prefer larger type.

### System Behavior
When a user enables large text mode:
- All text across the app scales up appropriately
- UI layout adjusts to remain readable and properly formatted at the larger text size
- No content is cropped, hidden, or broken by the text scaling

When a user disables large text mode:
- All text returns to the default size
- UI returns to its standard layout

The large text setting persists across sessions until the user changes it.

---

## High-Contrast Mode

### Description
Users can enable high-contrast mode to improve visual clarity across the app. In high-contrast mode, colors are adjusted to maximize distinction between text, icons, and background elements, making the interface easier to read and navigate.

### System Behavior
When a user enables high-contrast mode:
- UI colors are adjusted to a high-contrast palette
- Text and interactive elements remain clearly distinguishable from backgrounds and surrounding UI
- All screens and components respect the high-contrast color scheme

When a user disables high-contrast mode:
- The app returns to its default visual styling

The high-contrast setting persists across sessions until the user changes it.
